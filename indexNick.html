    This is the guide on how each and every indvidual attack Ryu uses works in SF5.
    Link to Street Fighter Page
    Link to Github Repository
    Hit a tab to get started on seeing data or read the legend below
    Each attack is catorgazied by the following keys:


    Neutral Attack or NA: An attack that happens when the character
    is stationary on the floor with no other movement inputs

    Forward Attack or FA:An attack that is activated as the character
    begins or is walking forward.

    Jumping Attack or JA:An attack that is activated as the character
    begins to jump or is in the air.

    Crouch Attack or CA:An attack that is activated as the character
    in a crouch stance.

    Each attack has an Damage Value, Stun Value, Hit Value

    Damage Value:Amount of damage an attack does which eventually leads to a K.O.

    Stun Value:A value that adds to a counter.When the counter reachs 1000
    the opposing character will be left stunned for more attacks.

    Hit Value:The amount of hits an attacks does on input
